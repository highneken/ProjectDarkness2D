# Project Darkness 2D

Noob-friendly 2D Action-Game (Godot 4) mit düsterem Environmental Storytelling.

## Aktueller Fokus (M1)
Spielbarer Kern: **Laufen + Springen + Tod/Respawn im Thronsaal**.

## Story-Kern
- Start: Erwachen auf dem Thron zwischen Leichen
- Fluch-Loop: Tod -> Respawn auf dem Thron
- Reveal: Protagonist ist das ultimative Böse
- Twist: Ausbruch führt in die moderne Welt

## Signature-Lore-Idee
**Kleine Mobs = verweste frühere Leichen des Protagonisten** (gescheiterte frühere Durchläufe).
