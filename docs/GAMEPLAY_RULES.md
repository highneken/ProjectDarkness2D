# GAMEPLAY RULES

## Core-Loop
1. Spawn im Thronsaal
2. Abschnitt erkunden und kämpfen
3. Gegner/Boss besiegen oder sterben
4. Bei Tod: Respawn auf Thron
5. Story-Schnipsel + neuer Versuch

## Player (MVP)
- Laufen (A/D, Pfeile)
- Springen (Space)
- Nahkampfangriff (J) [M2]
- HP-System [M2]

## Gegner (MVP)
- Enemy A: "Verwestes Echo" (Nahkampf-Läufer)
- Enemy B: später optional (Tank/Guard)
- Mini-Boss: mehr HP, 2 einfache Muster

## Death/Respawn
- Tod setzt Position auf Thronsaal-Spawn zurück
- Gegner werden zurückgesetzt
- Keine harte Progress-Strafe außer erneuter Lauf

## Schwierigkeit
- Fair, aber fordernd
- Lesbare Trefferzonen
- Erst klare Regeln, dann Komplexität
